# CR-CHIS
## Community Resilience Community Health Information System

CR-CHIS is a full-stack community health information system designed to support household-level health monitoring, field operations, follow-up workflows, and community-based health interventions.

The system is built to support community health workers, supervisors, NGOs, and health programs operating in low-resource and field-based environments.


==================================================
PROJECT VISION
==================================================

CR-CHIS aims to strengthen community-level health data collection and operational monitoring through a simple, scalable, and field-friendly digital platform.

The system focuses on:

• Household-centered health tracking  
• Vulnerability identification  
• Community follow-up workflows  
• Risk monitoring and alerts  
• Longitudinal health data management  


==================================================
CORE FEATURES
==================================================

HOUSEHOLDS MODULE

• Register households  
• Store demographic information  
• Track household location and size  


HOUSEHOLD MEMBERS MODULE

• Register household members  
• Track age and gender  
• Monitor vulnerable populations  
• Identify pregnant women  
• Identify persons with disabilities  
• Identify chronic illness cases  


VISITS MODULE

• Record household visits  
• Capture health observations  
• Monitor referrals  
• Track follow-up needs  


ALERTS MODULE

• Create health alerts  
• Track risks and interventions  
• Resolve alerts  
• Support operational monitoring workflows
